# 📌 Project Overview

**Saubhagya Stores** is a modern full-stack E-Commerce web application developed using **React.js**, **Spring Boot**, and **SQL**.  
The project provides a responsive user interface, secure backend APIs, product management features, and payment integration.

---

# 🚀 Features

✅ Fully Responsive UI Design  
✅ Product Add / Update / Delete Functionality  
✅ Dynamic Product Listing  
✅ RESTful APIs using Spring Boot  
✅ Razorpay Payment Gateway Integration  
✅ SQL Database Integration  
✅ Unit Testing with JUnit  
✅ Clean and Modern UI  
✅ Scalable Backend Architecture
